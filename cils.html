Цикл for

Этот цикл используется для того чтобы что-то несколько раз повторять вот код 

        
            for i in range(4):
                print("Hi")

Функция range задает сколько раз будет повторяться одно действие

Ещё есть цикл while True: этот цикл делает что-то бесконечно! его остановить только командой ctr + c или закрыть терминал вот пример 

			
				while True:
					print("hi")
