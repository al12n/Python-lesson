Математические операции

В этом уроке я вас познакомлю с математическими операциями на python начнем с баз

+ - складывает два числа не работает с строками только если использовать str

- - вычитает два числа тоже не работает со строками

* - умножает два компонента не работает на строках

/ - делит два компонента не работает на строках

@ - матричное умножение

** - делает число в степень пример: 5 ** 2 = 25

Это все математические операции которые вам надо знать
