Функции

И так в этом уроке я вас научу создавать свои функции. Для вас это 100% будет открытием что в программирование можно создать свои функции

В Python функции создаются командой def

Вот пример кода с своей функции

        
            def my_func(x, y):
                out = x * y
                return out

            my_func(1, 5)

И так давайте я вам здесь объясню в скобках мы показываем задаем две переменные делаем переменную которая умножает их и командой return показываем нашу переменную

Если вам что-то не понятно я вам крайне советую обратиться к ChatGPT и дать подробности чтобы вам было понятно

Есть ещё один способ создания функции это функция lambda Вот пример 

        
            my_func = lambda x, y: x * y

И так как вы заметили создание функции через lambda в одну строку это если вам нужно сделать маленькую функцию и все
