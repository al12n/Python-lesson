Random

И так библиотека random она самая классическая как импортируется библиотека вот как 

        
            import random

Если название слишком длинное можно изменить название командой as

        
            import random as rd

Так будет намного удобнее

Сейчас я вам покажу основные функции с random

random.randint(a, b) — возвращает случайное целое число от a до b включительно. Пример: random.randint(1, 6) — может вернуть любое число от 1 до 6.

random.random() — возвращает случайное число с плавающей точкой от 0.0 до 1.0.Пример: random.random() → 0.2746

random.choice(seq) — возвращает случайный элемент из последовательности (списка, строки и т.п.).

Пример: random.choice(['apple', 'banana', 'cherry']) → 'banana'

random.shuffle(list) — перемешивает список случайным образом. Пример:

        
            fruits = ['apple', 'banana', 'cherry']
            random.shuffle(fruits)
            print(fruits)

random.sample(population, k) — возвращает список из k уникальных элементов из последовательности.

Math

Так вы узнали библиотеку random теперь другая не меняя важная это math

Вот сейчас я вам покажу функции с math

math.sqrt(x) — квадратный корень из x Пример: math.sqrt(25) → 5.0

math.pow(x, y) — возводит x в степень y (аналог x ** y) Пример: math.pow(2, 3) → 8.0

math.floor(x) — округляет число вниз до ближайшего целого Пример: math.floor(3.9) → 3

math.ceil(x) — округляет число вверх Пример: math.ceil(3.1) → 4

math.pi — значение числа π (пи) Пример: math.pi → 3.141592...

math.e — число Эйлера (основание натурального логарифма) Пример: math.e → 2.71828...

math.sin(x), math.cos(x), math.tan(x) — тригонометрические функции (аргумент — в радиана) Пример: math.sin(math.pi / 2) → 1.0

math.log(x, base) — логарифм числа x по основанию base (если не указать base, по умолчанию — натуральный логарифм)

Пример: math.log(100, 10) → 2.0
