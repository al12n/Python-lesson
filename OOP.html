ООП

В этой главе я расскажу про ООП создание объектов вот пример ООП

        
            class Cat:
                def __init__(self, name, age): # создаем конструктор с нужными нам переменными
                    self.name = name
                    self.age = age
                    print("Name:" + name + ". Age:" + str(age))

            cat = Cat("Simba", 2)

Выглядит страшно, но вы поймете.

Значит мы создали объект с помощью команды class и кстати self это используется в классах Python надеюсь вы меня хорошо поняли
